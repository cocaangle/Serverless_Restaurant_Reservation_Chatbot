Chatbot creating: AWS Lambda, Amazon Lex, DynamoDB, CloudWatch <br>
Integrate to own website: AWS Cognito, AWS SDK, AWS S3 <br>
Integrate to multi-media channel(fb, slack): webhook, postback url, OAuth handshake <br>
